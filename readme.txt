Adds additional notification which can be used for automatic authority registrations (and cancellations of registrations)

== Description ==
This plugin extends WooCommerce und WooCommerce Bookings with the option to add a legal document number (decision number) to any product and to automatically sent a notification with important details to a legal public authority. This is especially useful for stores with bookable items for events for example. Since e-mails can be fully customized it can also be used for any other use case where a notification needs to be sent at a paid booking and another on cancellation.

== Installation ==
Go to Plugins > Add New > Upload  and select the zip file of the WordPress plugin you wish to add.
